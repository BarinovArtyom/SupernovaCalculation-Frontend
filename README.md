# Supernova Calculation

Веб-сервис для астрофизического анализа сверхновых типа Ia. Сервис предоставляет инструменты для расчета ключевых параметров вспышки: полной выделившейся энергии и массы синтезированного никеля-56. Расчеты выполняются на основе загруженных пользователем фотометрических данных, полученных с различных типов телескопов. Все результаты и история запросов сохраняются в личном кабинете пользователя.

### Ветки
- **React**: Клиентское одностраничное приложение, реализующее интерфейс гостя.
- **Redux**: Внедрение Redux Toolkit в React-приложение для хранения и управления состоянием фильтров страницы услуг.
- **gh-pages**: Развертывание собранной статической версии фронтенд-приложения.
- **Tauri**:  Версия клиентского интерфейса, упакованная в нативное desktop-приложение.
- **observatory-interface**: Интерфейс пользователя.
- **astronomer-interface**: Интерфейс модератора.
- **Asynch**: Реализация асинхронного межсервисного взаимодействия.
